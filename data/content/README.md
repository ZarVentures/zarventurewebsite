# Content Management

This directory contains JSON files for all website pages. You can edit these JSON files to update the content on your website without modifying the code.

## Available Content Files

- `home.json` - Home page content
- `about.json` - About page content
- `contact.json` - Contact page content
- `core-competencies.json` - Core Competencies page content
- `channel-partner.json` - Channel Partner page content
- `international-realty.json` - International Realty page content
- `rera-partners.json` - RERA Partners page content
- `privacy.json` - Privacy Policy page content
- `terms.json` - Terms & Conditions page content
- `rera-compliance.json` - RERA Compliance page content

## How to Update Content

1. Open the JSON file for the page you want to update
2. Edit the text content as needed
3. Save the file
4. The changes will be reflected on the website automatically

## Adding New Content

You can add new fields to the JSON files. The structure is flexible - just make sure to maintain the existing structure for fields that are already being used.
